Protein-Protein Interactions (PPI)

Overview

In addition to kinase-substrate interactions related to the KEA3 project, this website will also host data related to protein-protein interactions. Data was collected from eight different PPI databases, and all data was converted into the .sig file format. These .SIG files were later processed into the .GMT file format. The methods used to transform the original data and into the .sig and .gmt files for the protein-protein interactions can be found in the Jupyter Notebooks linked directly below.

Methods

There are five Jupyter Notebooks (which each process all of the databases in a single notebook) detailing the methods used to process the data. The 'PPI-Human Data' notebook filters data related to PPI-Human Data protein-protein interactions for all of the PPI databases, and the 'PPI- Mouse Data' notebook filters data related to PPI-Mouse Data protein-protein interactions for all of the PPI databases. The 'PPI-Combined' notebook combines PPI-Human Data and PPI- Mouse Data data together for each PPI database, and the 'Create Sig File' notebook creates a .SIG file for each database using the combined data. Finally, the 'SIG_to_GMT' notebook takes the .SIG files and converts them to .GMT files. for the .GMT file, only proteins which had at least five interactions and no more than 2000 interactions were included in the .gmt files.
