Understanding The Basics



How do I download the raw data?
For most of the interactions databases, the name of the database in the data tables will also serve as a link to the original website. From there, you can directly download the raw data from the website. Some databases no longer exist, so data may not be available for download.



What is a GMT file?
GMT files are used to describe gene sets, as each row corresponds to a different gene set. Each row lists a name (for example, a kinase in kinase-substrate interactions), a description, and the genes in the gene set (in this example, all of the substrates that interact with the kinase). Gene sets must have a minimum of five genes and a maximum of 2000 genes in order to be included in the gmt file. Only mice and human samples from the original data are included in the GMT file.


What is a SIG file?
SIG files represent gene interaction networks, also listed by row. In each row, the columns represent: source name, source human accession, source mouse accession, source type, source location, target name, target human accession, target mouse accession, target type, target location, effect, type of interaction, and lastly PubMed IDs. Some of these fields may be filled with 'NA' if information is not available to fill the column. All gene interactions are included in the SIG file, regardless of whether the aggregated gene set has fewer than five or more than 2000 genes. Only mice and human samples from the original data are included in the SIG file.


What is a Jupyter Notebook?
A Jupyter Notebook is a document containing live code, which can be created using the Jupyter web application. The data processing for all of the databases is written in Python 3, and the Jupyter Notebook for each interaction database can be viewed by selecting the Jupyter icon in the data tables. There, the code for data processing should be clearly annotated by comments explaining the methods.




What do 'Filtered' and 'Unfiltered' refer to?
For protein-protein networks, two options are available to download: 'Filtered' or 'Unfiltered' files. The 'Filtered' GMT and SIG files remove data related to PMIDs that correspond to more than ten interactions, while the 'Unfiltered' GMT and SIG files include these itneractions regardless of PMID.
