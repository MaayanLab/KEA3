KEA3

Overview

KEA2 (Kinase Enrichment Analysis 2) is a tool which takes an input of a gene list and outputs a list of associated kinases. This tool was originally built in 2009 and is comprised of manually curated kinase-substrate interactions. Since 2009, many databases that were originally curated have expanded and new databases have emerged, so the purpose of the KEA3 project is to collect and process new data pertaining to kinase-substrate interactions and expand the KEA2 database. Thus far, seven databases have been processed, and the data from each has been converted into .GMT and .SIG files, found below. Additionally, links to download the raw data and date of retrieval are also listed.

Link to KEA2 website

Methods

In order to convert the raw data into a .GMT and.SIG file, Python scripts were written for each individual database, and the conversion into both file formats was accomplished in each single script. Jupyter Notebooks corresponding to these scripts can be found for each database and include comments detailing the function of each code block for increased clarity. When processing the data, only kinases which had at least five substrates and no more than 2000 substrates are included in the .GMT files below. However, the .SIG files include all kinase-substrate interactions.
