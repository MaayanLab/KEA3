KEA3

Overview

KEA2 (Kinase Enrichment Analysis 2) is a tool which takes an input of a gene list and outputs a list of associated kinases. This tool (KEA) was originally built in 2009 and is comprised of manually curated kinase-substrate interactions. Since 2009, many databases that were originally curated have expanded and new databases have emerged, so the purpose of the KEA3 project is to collect and process new data pertaining to kinase-substrate interactions and expand the KEA2 database.

Link to KEA2 website

Methods

To match the current file format of KEA2 data, all data downloaded from the following databases needed to be converted from their original format to the .gmt file format. To accomplish this, a script was written for each dataset in Python to process the data. Both these scripts and the .gmt files created are linked below according to its corresponding resource. Links to the download pages of the original datasets are also included, and only data pertaining to the human and mouse species was considered in the final .gmt files.
